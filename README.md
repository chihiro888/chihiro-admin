# Chihiro Admin & Chihiro Develop Kit (CDK)

# How to start?

## database & image server

Please run the command below to build the database and image server.

```
bash start_dev.bash
```

## frontend

```
cd admin
yarn install
yarn dev
```

http://localhost:3001/

## backend

```
cd backend
yarn install
VSCode > Run ad Debug > debug NestJS
```

http://localhost:9001/api
