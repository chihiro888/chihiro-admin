# Mysql Query Saver

## Initial Setting

###  install nodeenv for mac

```bash
brew install nodeenv
```

### create nodeenv

```bash
nodeenv --node=16.16.0 nenv
```

### setup nodeenv

```bash
. nenv/bin/activate
```

### install yarn
```bash
npm install -g yarn
```

## backend
```bash
yarn install
```
```bash
yarn start
```

## frontend
```bash
yarn install
```
```bash
yarn start
```
